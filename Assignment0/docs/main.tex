The flowchart seen in figure \ref{figure:assignment0:flowchart} describes the IsLeapYear method.
The method takes a year of type int as input.
The method first test if the given year is below or equal to 1582, if it is then throw an error.
There after we test if the year is divisible by 4, if not, then it's not a leap year.
Then test if the year is divisible by 100, if not then it's a leap year.
If it is divisible by 100, test if it's divisible by 400, if not then it's not a leap year. If it is divisible by 400 it's a leap year.
